# web-clone-project

How to run this file
===================================
1. Download the 'CCCC Web Clone Project.zip' file.
2. Unzip this zip file into your destination folder
3. Open the new folder you just extracted.
4. Choose language:
	1) English: Open the `en` folder, then double-click the `index.html` file inside it.
	2) Chinese: Open the `zh-MY` folder, then double-click the `index.html` file inside it.
5. This will open the project in your web browser.

Note: You can still switch languages at any time using the selector in the top-right corner of the webpage.


如何运行此文件
===================================
1. 下载‘CCCC Web Clone Project.zip’压缩文件。
2. 将此压缩文件解压到您的目标文件夹。
3. 打开您刚刚解压出来的新文件夹。
4. 选择语言：
	1) 英文版：打开 `en` 文件夹，然后双击里面的 `index.html` 文件。
	2) 中文版：打开 `zh-MY` 文件夹，然后双击里面的 `index.html` 文件。
5. 这将在您的网页浏览器中打开此项目。

备注：您仍然可以随时使用网页右上角的语言选择器来切换语言。
